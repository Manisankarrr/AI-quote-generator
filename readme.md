# ✨ AI-quote-generator

A simple yet powerful AI-powered quote generator built with [Streamlit](https://streamlit.io/). Instantly generate motivational and coding quotes with a modern web interface.

---

## 📚 Introduction

**AI-quote-generator** lets you generate motivational and coding quotes on demand! Powered by Streamlit, this web app provides a fun and interactive way to get inspired or share wisdom with friends and colleagues.

---

## 🚀 Features

- **Instant Quotes**: Get motivational and coding quotes instantly.
- **AI Integration**: (Optional) Enhance quotes with AI-generated content using OpenRouter API.
- **User-friendly UI**: Clean, interactive Streamlit interface.
- **Ready to Run**: Quickly spin up the app locally or in a development container.

---

## 🛠️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/yourusername/AI-quote-generator.git
cd AI-quote-generator
```

### 2. (Optional) Use Dev Container

If you use [GitHub Codespaces](https://github.com/features/codespaces) or [Dev Containers](https://containers.dev/):

- The project includes a pre-configured `.devcontainer/devcontainer.json` for a seamless Python development experience.

### 3. Install Dependencies

It's recommended to use a virtual environment.

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

Or, manually install:

```bash
pip install streamlit python-dotenv requests
```

### 4. (Optional) Set Up API Key

For AI-powered quote generation, create a `.env` file and add your [OpenRouter API Key](https://openrouter.ai/) as follows:

```env
OPENROUTER_API_KEY=your_api_key_here
```

---

## ▶️ Usage

Run the Streamlit app:

```bash
streamlit run app.py
```

Then open your browser at [http://localhost:8501](http://localhost:8501) to start generating quotes!
